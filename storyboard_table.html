10-Step User Experience Storyboard

Step User Action System Response Visual Feedback
1
 Opens app in browser Loads "AI Coach" interface Purple gradient header, empty chat area, green "Start Chat" button

 Clicks "Start Chat" Requests microphone permission Button shows loading spinner, status: "Connecting..."
3
 Clicks "Allow" microphone Connects to Gemini AI Status: "Listening...", red "Stop Chat" button, mic icon active
4
 Speaks: "Hello, can you hear me?" Real-time transcription starts Blue message bubble appears on right, text streams live
5
 Stops speaking (pause detected) Finalizes user message, AI processes User bubble completes, thinking indicator appears
6
 Waits for AI response AI generates text response Grey bubble on left, text streams word-by-word
7
 AI text completes Audio playback begins Speaker icon animates, bubble highlighted during audio
8
 Responds: "Tell me more" Cycle repeats: transcribe → process → respond New messages stack upward, auto-scroll to latest
9
 Continues conversation Chat history builds up Scrollable area, clear user/AI message distinction
10
 Clicks "Stop Chat" Ends session gracefully Button back to green "Start Chat", status: "Idle"
